At the time of writing, the latest version of OpenShift is 4.1.14 and this is reflected in the sample download instructions below.

# RHCOS

Download the RHCOS OVA for OCP 4.1 from https://mirror.openshift.com/pub/openshift-v4/dependencies/rhcos/4.1/latest/rhcos-4.1.0-x86_64-vmware.ova

To use this OVA for master nodes in the cluster, set the variable master_ova_path to the path for the RHCOS OVA file. The OVA used by worker nodes is determined by the worker_ova_path variable and will default to the same OVA as that used for master nodes.

As part of the installation process, templates will be created from the master and worker OVAs. These templates will be named based on the variables master_template and worker_template.

# Clients

The latest version of the client software is available at https://mirror.openshift.com/pub/openshift-v4/clients/ocp/latest/

The openshift-install software used by the playbooks to install the cluster can be downloaded from https://mirror.openshift.com/pub/openshift-v4/clients/ocp/latest/openshift-install-linux-4.1.14.tar.gz

Download and unpack the file and then set the variable ocp_installer_path in your configuration file to the path for the openshift-install program.

Two client programs are used to interact with a deployed cluster - oc and kubectl. These programs are available in a single file at https://mirror.openshift.com/pub/openshift-v4/clients/ocp/latest/openshift-client-linux-4.1.14.tar.gz

Download and unpack the file. Set the variable ocp_oc_path in your configuration file to the path for the ocp program and set ocp_kubectl_path to the path for the kubectl program.

# Downloading and unpacking

An example of the commands required to download and unpack the required software is shown below:

$ mkdir ~/kits
$ cd ~/kits
$ wget https://mirror.openshift.com/pub/openshift-v4/clients/ocp/latest/openshift-client-linux-4.1.14.tar.gz
$ wget https://mirror.openshift.com/pub/openshift-v4/clients/ocp/latest/openshift-install-linux-4.1.14.tar.gz
$ wget https://mirror.openshift.com/pub/openshift-v4/dependencies/rhcos/4.1/latest/rhcos-4.1.0-x86_64-vmware.ova
$ tar -xvf openshift-client-linux-4.1.14.tar.gz
$ tar -xvf openshift-install-linux-4.1.14.tar.gz
